Wikidata Dump

A Wikidata dump of the Neo4J database. Be aware it has a size of ~115GB. It's SHA1 checksum is 305e53cc3cb7fdba2cea08d69fe6b354bb78eaab. If you're running Neo4j Desktop follow the instructions at https://neo4j.com/docs/desktop-manual/current/operations/create-from-dump/ to create a database instance from the dump. If you're running neo4j either directly on your machine or via docker use the neo4j-admin database load command.
